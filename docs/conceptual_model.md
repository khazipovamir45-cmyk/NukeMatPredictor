# Концептуальная модель

## 1. Общая идея системы

NukeMatPredictor — это система подбора материалов для ядерной инженерии на основе заданных пользователем условий эксплуатации.

Пользователь вводит параметры среды (температура, облучение и т.д.), а система:
- анализирует базу материалов
- находит наиболее подходящие
- ранжирует их (оптимальный, допустимый, неподходящий)


## 2. Используемые признаки

В модели используются 5 основных признаков:

1. **Температура (temperature, °C)**  
   Условия эксплуатации материала.

2. **Минимальная требуемая прочность (min_required_strength, MPa)**  
   Минимально допустимая прочность материала.

3. **Доза облучения (irradiation_dose, dpa)**  
   Влияние радиации на материал.

4. **Теплопроводность (thermal_conductivity, W/m·°C)**  
   Способность материала проводить тепло.

5. **Теплоёмкость (heat_capacity, J/kg·°C)**  
   Способность материала аккумулировать тепло.


## 3. Алгоритм k-NN

В проекте используется алгоритм **k ближайших соседей (k-NN)**.

### Принцип работы:

1. Все материалы представляются как точки в 5-мерном пространстве признаков
2. Данные нормализуются с помощью `StandardScaler`
3. Для введённых пользователем параметров:
   - вычисляется расстояние до всех материалов
   - выбираются k ближайших (k = 5)

### Расстояние:

Используется **евклидово расстояние**:
distance = sqrt((x1 - y1)^2 + ... + (x5 - y5)^2)


### Результат:

- ближайший - **оптимальный**
- второй - **допустимый**
- самый дальний - **непригодный**


## 4. Физическая модель

### 4.1 Расчёт базовой прочности (strength_ref)

Прочность материала при нормальных условиях рассчитывается как:

strength_ref = min_required_strength
+ temp_coef * max(0, temperature - 20)
+ dose_coef * irradiation_dose


Где:
`temp_coef` — коэффициент влияния температуры
`dose_coef` — коэффициент влияния радиации


### 4.2 Зависимость прочности от дозы

strength = strength_ref - dose_coef * log(1 + dose)

### 4.3 Зависимость прочности от температуры

strength = strength_ref - temp_coef * (T - 20)^1.2

### 4.4 Коррозия

Параметр:

corrosion_rate (mm/year)

Используется как характеристика деградации материала со временем.

## 6. Поток работы системы

1. Пользователь вводит параметры
2. Данные нормализуются
3. Выполняется k-NN поиск
4. Выбираются материалы:
   - оптимальный
   - допустимый
   - неподходящий
5. Строятся графики зависимостей
6. Результат выводится пользователю
