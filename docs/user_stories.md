# User Stories

Документ описывает пользовательские истории проекта **NukeMatPredictor**.

Формат: «Как [роль], я хочу [действие], чтобы [ценность]».
К каждой истории прилагается чеклист **Definition of Done (DoD)** — критерии приёмки, по которым считается, что история реализована полностью.

---

## US-1. Подбор материала под условия эксплуатации

**Как** студент ядерного направления,
**я хочу** ввести параметры эксплуатации (температуру, дозу облучения, минимальную прочность, теплопроводность, теплоёмкость),
**чтобы** быстро подобрать подходящий материал под заданные условия без ручного перебора справочников.

**Definition of Done:**
- [ ] На странице формы доступны 5 полей ввода: температура (°C), доза облучения (dpa), минимальная прочность (МПа), теплопроводность (Вт/м·°C), теплоёмкость (Дж/кг·°C).
- [ ] Все поля являются обязательными — при пустом поле форма не отправляется и пользователь видит подсказку.
- [ ] После отправки формы пользователь попадает на страницу результатов в течение 3 секунд.
- [ ] На странице результатов отображается таблица из 3 материалов со статусами «оптимальный», «допустимый», «непригодный».

---

## US-2. Изучение характеристик материалов и графиков

**Как** аспирант / научный сотрудник,
**я хочу** видеть характеристики подобранных материалов и графики зависимостей их свойств,
**чтобы** обосновать выбор материала для научной работы или отчёта.

**Definition of Done:**
- [ ] Для каждого материала в таблице отображаются: название, прочность, теплопроводность, теплоёмкость, коэффициент запаса, статус.
- [ ] Для оптимального материала строится график зависимости прочности от дозы облучения.
- [ ] Для оптимального материала строится график зависимости прочности от температуры.
- [ ] У каждого графика подписаны оси с единицами измерения.

---

## US-3. Сохранение результатов расчёта

**Как** пользователь,
**я хочу** сохранить результаты расчёта на устройство,
**чтобы** использовать их в отчёте или презентации.

**Definition of Done:**
- [ ] На странице результатов есть кнопка сохранения / экспорта.
- [ ] В сохранённый файл попадают введённые параметры, таблица материалов и графики.
- [ ] Файл открывается стандартными средствами ОС без дополнительного ПО.

---

## US-4. Интерактивное исследование зависимости от температуры

**Как** инженер-материаловед,
**я хочу** изменять температуру через интерактивный элемент на графике,
**чтобы** оценить поведение материалов в разных диапазонах температур без перезапуска расчёта.

**Definition of Done:**
- [ ] На графике есть интерактивный слайдер или поле ввода температуры.
- [ ] При изменении значения график перерисовывается без перезагрузки страницы.
- [ ] Допустимый диапазон температур указан явно (минимум и максимум).
- [ ] Текущее числовое значение температуры отображается рядом с элементом управления.

---

## US-5. Интерпретация результатов модели

**Как** студент ядерного направления,
**я хочу** понимать, почему конкретный материал помечен как оптимальный, допустимый или непригодный,
**чтобы** научиться правильно интерпретировать результаты модели.

**Definition of Done:**
- [ ] Рядом со статусом материала отображается короткое пояснение (1–2 фразы).
- [ ] В пояснении указаны конкретные числовые значения, повлиявшие на статус (например, «прочность 350 МПа выше требуемой 300 МПа»).
- [ ] Если материал помечен как «непригодный», явно указан параметр, по которому он не прошёл.
