# Non-Functional Requirements

## NFR-1. Время обработки запроса
Система должна обрабатывать запрос пользователя (подбор материалов) за разумное время.

## NFR-2. Скорость загрузки страниц
Страницы должны загружаться без заметных задержек.

## NFR-3. Стабильность работы
Система не должна падать при обычном использовании.

## NFR-4. Интуитивность интерфейса
Интерфейс должен быть понятен без инструкции.

## NFR-5. Простота процесса подбора
Пользователь должен выполнить подбор материалов за несколько шагов:  
1) ввести параметры, 2) нажать кнопку, 3) получить результат.

## NFR-6. Подписанность полей ввода
Все поля должны быть подписаны.

## NFR-7. Отсутствие установки дополнительного ПО
Не требуется установка дополнительного ПО.

## NFR-8. Защита от некорректного ввода
Нельзя вводить некорректные значения.

## NFR-9. Модульность кода
Код должен быть разбит на файлы.

## NFR-10. Читаемость кода
Код должен быть читаемым и понятным.

## NFR-11. Доступность через браузер
Система должна быть доступна через браузер.
