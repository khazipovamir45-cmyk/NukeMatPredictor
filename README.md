# NukeMatPredictor

Веб-сайт для прогнозирования изменения прочности конструкционных материалов под воздействием нейтронного облучения и высоких температур.

---

## О проекте

**NukeMatPredictor** — это информационно-расчетный веб-сайт, разработанный в рамках учебного курса. Сайт позволяет специалистам в области ядерной энергетики и материаловедения выполнить расчет на основе введённых параметров.

---

## Функциональность

- **Главная страница** — описание проекта и кнопка перехода к расчёту.
- **Страница формы** — ввод параметров расчёта (температура, доза облучения и так далее).
- **Страница результатов** — таблица материалов с рассчитанной прочностью и коэффициентом запаса, график зависимости прочности от дозы облучения.
- **База данных** — 98 материалов с характеристиками.

### Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Установка и запуск

**1. Клонировать репозиторий:**
```bash
git clone https://github.com/ваш-username/NukeMatPredictor.git
cd NukeMatPredictor
```

**2. Создать и активировать виртуальное окружение:**

