NukeMatPredictor

Веб-сайт для прогнозирования изменения прочности конструкционных материалов под воздействием нейтронного облучения и высоких температур.

О проекте

NukeMatPredictor — это информационно-расчетный веб-сайт, разработанный в рамках учебного курса. Сайт позволяет специалистам в области ядерной энергетики и материаловедения ознакомиться с методологией оценки остаточной прочности материалов после облучения и выполнить расчет на основе введённых параметров.

Функциональность

- Главная страница — описание проекта и кнопка перехода к расчёту.
- Страница формы — ввод параметров расчёта (температура, доза облучения, среда).
- Страница результатов — таблица материалов с рассчитанной прочностью и коэффициентом запаса, график зависимости прочности от дозы облучения.
- База данных — 98 материалов с характеристиками.
- Сортировка — возможность сортировки таблицы по любому столбцу.
